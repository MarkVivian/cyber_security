To format text in a README.md file, you can use Markdown syntax.
Markdown is a lightweight markup language that allows you to add formatting to plain text.

    1) OPTIONS ♟️

    BOLD TEXT : To make text bold:
            i) surround it with double asterisks ** e.g  **bold text** 
           ii) double underscores __. e.g.  __bold text__.

    ITALICS TEXT: To make text italic:
            i) surroud it with single asterisks * e.g. *italics text*
           ii) surround it with single underscores _. e.g. _italic text_.

    BOLD AND ITALICS TEXT: To make text bold and italic:
            i) surround it with triple asterisks *** e.g. ***bold and italic text*** 
            ii) triple underscores ___. e.g.  ___bold and italic text___.

    CODE : To display code or inline code snippets:
            i) use backticks `. For inline code, surround the code with single backticks.
            ii) For code blocks, use triple backticks before and after the code. 

    HEADERS : To create headers, use hash symbols #. The number of hash symbols determines the level of the header
            i) # Header 1
            ii) # Header 2
            iii) # Header 3

    Lists: To create ordered and unordered lists,
            i) use hyphens - 
                a) - Item 1
                    - Item 2
                        - Subitem 1
                        - Subitem 2
            ii) numbers followed by periods .. 
                a)1. Item 1
                  2. Item 2

    Links: To add links
            i)use square brackets [] for the link text, followed by parentheses () containing the link URL. For example: [Link Text](http://example.com).

    Images: To embed images:
            i) use an exclamation mark ! followed by square brackets [] for the image alt text, followed by parentheses () containing the image URL or path. For example: ![Alt Text](path/to/image.jpg).
